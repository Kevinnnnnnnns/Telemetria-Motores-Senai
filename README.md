# Telemetria-Motores-Senai
Projeto: Sistema de Telemetria e Monitoramento em Tempo Real para Ensaios de Motores Elétricos
Este projeto consiste em um protótipo de telemetria industrial desenvolvido para modernizar a coleta de dados em postos de teste de motores. Utilizando um ESP32, o sistema realiza a leitura automatizada de temperatura (DHT22), vibração e corrente elétrica (simulados por potenciômetros). Os dados são transmitidos via protocolo MQTT para uma central de monitoramento desenvolvida em Java (Maven), garantindo a integridade das informações e substituindo registros manuais obsoletos.
